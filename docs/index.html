ParkDude

ParkDude is a parking space managment system which employees can use to reserve parking spaces. With the application users can see available parking spaces, reserve them or mark them as temporarily free if they are their permanent owners.

The system is primarily used with mobile application, targeted for Android and iOS platforms. The system also has a web based admin panel and a Slack integration.

ParkDude project is done as part of Tampere University's TIE-13106 Project Work on Pervasive Systems course. The project is done for Innogiant.
